1. Variable Declaration in JS

Declaring a variable means reserving memory to store data.

JS keywords for declaration:

var → old, redeclarable, function-scoped

let → ES6+, block-scoped, can reassign but not redeclare in the same scope

const → ES6+, block-scoped, cannot reassign or redeclare

Naming rules:

No spaces

Cannot start with a number

Avoid kebab-case (my-variable)

Preferred: camelCase, snake_case, PascalCase

2. Data Types

Primitive Types (immutable)

Number → integers, floats

BigInt → for very large numbers → 123456n

String → "Hello" or 'Hello'

Boolean → true / false

Null → empty value

Undefined → declared but not initialized

Non-Primitive Types (mutable)

Array → [1,2,3]

Object → {name:"Jagath", age:24}

Array of objects → [ {name:"A"}, {name:"B"} ]

Function → function greet(){...}

3. Printing Statements

console.log("Hello")  // prints in console

alert("Hello")        // popup in browser

document.write("Hello") // writes to HTML page

4. Syntax Example

let userName = "Jagath"; // declaration + initialization
const pi = 3.14;         // cannot reassign(standard values only example pi=3.14,week=7,year=365 etc...)
var age;                  // declaration only, default = undefined

5. Mutability

Keyword	Redeclare	Reassign	Scope
var	✅	✅	Function
let	❌	✅	Block
const	❌	❌	Block

6. Task Example

You want 10 variables, 3 different data types, and reassign them. Here’s one way:

// Numbers
var num1 = 10;
let num2 = 20;
const num3 = 30;

// Strings
var name1 = "Jagath";
let name2 = "React";
const name3 = "JavaScript";

// Boolean
var isActive = true;
let isLoggedIn = false;
const hasAccess = true;

// Reassigning
num1 = 15;   // var can reassign
num2 = 25;   // let can reassign
// num3 = 35; // const cannot reassign -> will give error

name1 = "Angular"; // var
name2 = "NodeJS";  // let
// name3 = "Python"; // const -> error

isActive = false;  // var
isLoggedIn = true; // let
// hasAccess = false; // const -> error

console.log(num1, num2, num3);
console.log(name1, name2, name3);
console.log(isActive, isLoggedIn, hasAccess);


✅ This covers:

10 variables

3 data types: Number, String, Boolean

Reassigning allowed for var & let, blocked for const
