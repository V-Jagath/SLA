🔑 3 Types of Variable Keywords in JavaScript

var

Declaration → Allowed

Initialization → Allowed

Re-assign → Allowed

Scope → Function scope (old way, not recommended)

let

Declaration → Allowed

Initialization → Allowed

Re-assign → Allowed

Scope → Block scope (recommended for values that may change)

const

Declaration → Allowed

Initialization → Must be initialized at declaration

Re-assign → ❌ Not allowed

Scope → Block scope (used for fixed values)

⚡ Operators (4 Main Types = AACL)

1. Arithmetic Operators

+ (Addition)

- (Subtraction)

* (Multiplication)

/ (Division)

% (Modulus → remainder)

** (Exponentiation → power)

++ (Increment → x++ = x+1)

-- (Decrement → x-- = x-1)

👉 Example:

let x = 5;
x++;   // 6
x**2;  // 36

2. Assignment Operators

= → Assign

+= → Add & assign

-= → Subtract & assign

*= → Multiply & assign

/= → Divide & assign

%= → Mod & assign

**= → Exponentiation & assign

👉 Example:

let total = 500;
total += 50;   // 550

3. Comparison Operators

== → Equal to (loose, ignores type)

=== → Strict equal (checks value + type)

!= → Not equal

!== → Strict not equal

< → Less than

<= → Less than or equal to

> → Greater than

>= → Greater than or equal to

👉 Example:

5 == "5";   // true
5 === "5";  // false

4. Logical Operators

&& (AND)

true && true → true

otherwise → false

|| (OR)

false || true → true

true || false → true

true || true → true

false || false → false

! (NOT)

!true → false

!false → true

👉 Example:

let a = true, b = false;
a && b;  // false
a || b;  // true
!a;      // false
